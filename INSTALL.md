# Voice Flow 安装说明

## 系统要求

- Windows 10 / 11（64 位）
- 麦克风设备
- 网络连接

---

## 安装步骤

1. 双击 `VoiceFlow-Setup.exe`，按提示完成安装
2. 安装完成后，桌面和开始菜单会出现 Voice Flow 快捷方式

---

## ⚠️ 重要：杀毒软件误报说明

Voice Flow 使用业界标准的 Python 打包技术（PyInstaller），部分杀毒软件会将其**误判为威胁**。这是打包工具的特征被恶意软件滥用的行业通病，**并非软件本身有问题**。

如果遇到以下拦截，请按对应步骤操作：

---

### Windows Defender（Windows 自带）

**现象：** 下载时提示"此文件可能危害你的计算机"，运行时 SmartScreen 显示"Windows 已保护你的电脑"

**操作步骤：**

1. **下载被拦截时：**
   - 在 Chrome/Edge 下载栏 → 右键被拦截的文件 → 选择 **"保留"**
   - 或点击 `...` → **"保留"** → **"仍然保留"**

2. **运行时 SmartScreen 拦截：**
   - 点击 **"更多信息"**
   - 点击 **"仍要运行"**

3. **安装后如被 Defender 隔离，添加排除项：**
   - 打开 **开始菜单** → 搜索 **"Windows 安全中心"**
   - 点击 **"病毒和威胁防护"**
   - 点击 **"管理设置"**（在"病毒和威胁防护设置"下方）
   - 向下滚动到 **"排除项"** → 点击 **"添加或删除排除项"**
   - 点击 **"添加排除项"** → 选择 **"文件夹"**
   - 浏览并选中 Voice Flow 安装目录（默认 `C:\Program Files\VoiceFlow`）
   - 点击 **"选择文件夹"** 完成

---

### 360 安全卫士

**现象：** 下载后自动删除安装包，或运行时提示"发现木马"

**操作步骤：**

1. **恢复被删除的安装包：**
   - 打开 360 安全卫士 → 点击 **"木马查杀"**
   - 点击左下角 **"恢复区"**
   - 找到 `VoiceFlow-Setup.exe` → 勾选 → 点击 **"恢复"**
   - 在弹出的确认框中勾选 **"恢复后信任此文件，不再查杀"** → 确定

2. **添加信任目录（安装后）：**
   - 打开 360 安全卫士 → 右上角 **"设置"**（三条横线图标）
   - 点击 **"文件白名单"**
   - 点击 **"添加目录"** → 选择 Voice Flow 安装目录 → 确定

---

### 火绒安全

**现象：** 运行时弹出"发现病毒"窗口

**操作步骤：**

1. 在弹出的火绒窗口中 → 点击 **"暂不处理"**
2. 勾选 **"记住我的选择"**
3. 或者主动添加信任：
   - 打开火绒 → 右上角菜单 → **"信任区"**
   - 点击 **"添加文件夹"** → 选择 Voice Flow 安装目录

---

### 腾讯电脑管家

**现象：** 下载后报毒并自动删除文件

**操作步骤：**

1. 打开电脑管家 → 点击 **"病毒查杀"**
2. 点击 **"隔离区"** → 找到 `VoiceFlow-Setup.exe`
3. 点击 **"恢复"** → 勾选 **"信任"**
4. 安装后：设置 → **"信任管理"** → 添加 Voice Flow 安装目录

---

## 验证安装

安装并启动后，您应该看到：

1. 桌面右下角出现 **Voice Flow 托盘图标**
2. 首次启动弹出 **设置对话框**（配置 API 密钥）
3. 如果没有弹出设置，点击托盘图标 → 选择 **"设置"**

## 常见问题

**Q: 为什么一定要添加排除项？**

A: 杀毒软件会将 Voice Flow 的加密代码误判为病毒。添加排除项后，软件才能正常运行。Voice Flow 不会收集您的个人信息，也不会访问任何与语音转文字无关的文件。

**Q: 添加排除项安全吗？**

A: 安全。排除的是 Voice Flow 的安装目录，不是整个系统。如果您从官方渠道获取安装包，软件经过加密保护，不会被其他程序篡改。

**Q: 软件更新后需要重新设置吗？**

A: 覆盖安装在同一目录下，不需要重新添加排除项。如果安装了新路径，需要重新添加。

**Q: 试用期多久？**

A: 7 天免费试用，试用期间功能完整。到期后需购买许可证密钥激活。
