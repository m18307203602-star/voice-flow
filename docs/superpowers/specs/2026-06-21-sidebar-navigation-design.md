# Voice Flow 侧边栏导航 + 统计模块 — 设计规范

## 目标

将当前上下堆叠的单页布局改为 **左窄右宽** 的导航式布局，左侧 QListWidget 做导航，右侧 QStackedWidget 切换四页内容。

---

## 整体布局

```
┌──────────┬─────────────────────────────────────────────┐
│          │                                              │
│  🏠 首页  │  首页内容（所有现有控件 + 最近记录快捷预览）    │
│          │                                              │
│  📋 历史  │  ← QStackedWidget，根据左侧选中切换            │
│          │                                              │
│  📖 词典  │                                              │
│          │                                              │
│  📊 统计  │                                              │
│          │                                              │
└──────────┴─────────────────────────────────────────────┘
```

### 左侧导航栏

- 组件：`QListWidget`（自带选中高亮、悬停效果）
- 宽度：固定 180px
- 样式：深色背景 `#12121a`，选中项紫色左竖条 `#7c5cfc`，文字 `#e4e4f0`
- 四个导航项：🏠 首页 · 📋 历史记录 · 📖 词典 · 📊 统计

### 右侧内容区

- 组件：`QStackedWidget`
- 四页切换，跟随左侧选中项联动

---

## 四页内容定义

### 1. 首页（index 0）

保留当前所有控件，从上到下：

1. **顶部配置行** — 模式下拉 + STT引擎选择 + STT短连接模式 + 主模型下拉 + 设置按钮
2. **控制栏** — 录音按钮 + 对比按钮 + 状态标签 + 电平条
3. **引擎状态行** — 腾讯/讯飞/阿里状态 + 用量统计按钮
4. **模型对比面板**（默认隐藏，录音对比时展开）
5. **声纹浮动窗口** — `VoiceprintWidget`，定时器驱动
6. **最近记录** — 5 条历史记录快捷预览（带时间戳和文本摘要），点击可跳转到"历史记录"页
7. **法律声明滚动字幕** — 底部固定

首页不嵌入完整的 `HistoryPanel`，只用简化列表显示最近 5 条。

### 2. 历史记录（index 1）

- 嵌入现有 `HistoryPanel`，占满整个右侧区域
- 历史面板自带搜索、时间筛选、详情查看、导出功能
- 从首页"最近记录"点击某条 → 自动切到此页并定位

### 3. 词典（index 2）

- 将 `DictionaryDialog` 内容改为嵌入式 `QWidget`（新建 `dictionary_widget.py`）
- 内容：标题 + 启用开关 + 提示 + 表格 + 增删/导入/导出按钮
- 不再弹窗，直接在右侧内容区显示
- 保存按钮改为"保存"，不关闭（因为没有弹窗可关）

### 4. 统计（index 3）— 新增

三个核心指标卡片横排顶部：

| 卡片 | 内容 |
|------|------|
| 总口述时间 | 所有录音时长累加，格式 `X小时Y分钟` |
| 平均口述速度 | 总字数 ÷ 总时长，格式 `X 字/分钟` |
| 总录音次数 | 数字 |

下方两列：

- **左列：STT 引擎用量扇形图** — 腾讯/阿里/讯飞各占比例，`QPainter` 手绘
- **右列：过去 7 天趋势柱状图** — 每天录音次数 + 总时长，`QPainter` 手绘

数据来源：`HistoryDB`，启动时计算一次，切换到此页时刷新。

## 改动范围

### 文件清单

| 文件 | 操作 | 说明 |
|------|------|------|
| `ui/main_window.py` | 重构 ~250 行 | 布局从 `QVBoxLayout` 改为 `QHBoxLayout(sidebar + QStackedWidget)`，首页嵌入到 page 0 |
| `ui/sidebar.py` | **新建** | `Sidebar(QListWidget)`，封装导航样式和交互 |
| `ui/stats_page.py` | **新建** | 统计页，扇形图 + 柱状图 + 指标卡片 |
| `ui/dictionary_widget.py` | **新建** | 词典嵌入式 Widget，从 `dictionary_dialog.py` 提取 |
| `ui/dictionary_dialog.py` | 保留不动 | 作为外部弹窗的兼容入口（菜单栏仍可用） |
| `ui/history_panel.py` | 微调 ~10 行 | 新增 `jump_to_item(item_id)` 方法供首页跳转 |

### 不改动的文件

- `main.py` — 入口不变
- `config.py` — 配置不变
- `engine/*` — 业务逻辑不变
- `dictionary.py` — 数据模型不变
- `storage/history_db.py` — 数据库不变

---

## 数据流

```
HistoryDB ──→ stats_page.py（计算统计数据）
          ──→ history_panel.py（历史列表）
          ──→ main_window.py（首页最近5条预览）

DictionaryManager ──→ dictionary_widget.py（词库编辑）
                  ──→ session.py（STT 后替换）
```

---

## 样式规范

- 侧边栏背景 `#12121a`，选中项左边 3px `#7c5cfc` 竖条
- 侧边栏文字 `#e4e4f0`，图标用 emoji 前缀
- 统计卡片背景 `#1c1c2e`，圆角 10px，数值 `#cba6f7` 色
- 扇形图/柱状图用 QPainter 手绘，配色：腾讯 `#7c5cfc` / 阿里 `#f97316` / 讯飞 `#22c55e` / 其他 `#6b7280`
- 首页最近记录用紧凑列表，每条高度 ~40px，hover 变亮

---

## 验收标准

1. 左侧四个导航项可点击切换，选中项高亮
2. 首页功能完整：录音、对比、模式切换、主模型切换、声纹、最近记录
3. 历史记录页：搜索、翻页、详情、导出正常
4. 词典页：增删改查、导入导出、启用开关正常
5. 统计页：总口述时间、平均速度、录音次数、扇形图、柱状图数据正确
6. 窗口最小 800x600 时侧边栏不折叠
7. 热键录音功能不受影响（全局热键 + 取消）
